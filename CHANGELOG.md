# Changelog

All notable changes to awesomepapers-core will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added
- Initial release of awesomepapers-core theme
- Layouts: default, page, publication
- Includes: head, sidebar
- CSS: Hyde theme, Poole, Lanyon, syntax highlighting
- JavaScript: alp-chat, papers-worker, site-config
- Pages: papers, tags, author-viz, topic-viz, tsne-viz, digest, contributing, etc.
- Build scripts in etc/ directory
- Full configuration system via _config.yml
- Support for GitHub Pages remote theme
- Dark mode support
- Responsive design for mobile/tablet/desktop
- AI chat widget integration
- Email digest subscription system

### Features
- Paper search and filtering with multiple sort options
- Tag/topic exploration with nested taxonomy
- Author visualization and statistics
- t-SNE paper embeddings map
- "NEW" badge for recently added papers
- Citation counts, GitHub stars, Hugging Face likes display
- Full customization via site config
